1. 변수
    1) 값이 변하는 수
        int num = 1;

2. 상수
    1) 상수는 변하지 않는 값
    2) 종류
        - 리터럴(literal)
            - 문자 그대로의 값
            - 프로그램이 시작할 때 특정 메모리 공간 상수 풀(constant pool)에 자동으로 로딩됨
            - 우리가 변수의 값을 대입할 때, 상수 풀에서 복사하여 변수에 대입을 해주는 것
        - final 상수
            - 프로그래머가 임의로 만듬
            - final 상수 이름은 전부 대문자를 이용해서 만듬
                - ex) final int MAX_SIZE = 100
            - 단어 단위를 구분해서 읽기 쉽게 언더바 _를 이용하여 단어를 구분함
            - 처음 값을 정하게 되고 이후에는 다른 값을 대입하여 바꿀 수 없음
    
3. 자료형의 형변환(Type conversion)
    1) 형변환 표현
        
        주체                          용어
      -----------------------------------------------------------------
        컴파일러(가 자동으로)       자동 형변환     묵시적(암묵적) 형변환
        프로그래머(가 직접)         강제 형변환     명시적 형변환

    2) 자동 형변환의 방향 
        - 바이트 크기가 작은 자료형에서 큰 자료형으로의 형변환은 자동으로 이루어짐
            - byte => short => int => long
                      char
        - 덜 정밀한 자료형에서 더 정밀한 자료형으로의 형변환은 자동으로 이루어짐
            - byte => short => int => long => float => double
                      char

    3) 연산 시 형변환
        - 산술 연산시 형변환 발생함
            - 연산은 CPU가 담당함
                - 입력1     입력2       결과
                    int     int         int
                    long    long        long
                    float   float       float
                    double  double      double
